SqlStudio


Provider independent SQL query tool

Overview

Tool to execute SQL commands against a MSSQL, Oracle, ODBC or Sqlite database.

In the Input tab you can write queries folowed by a semicolon(;) and enter and the query will be executed on the curent database. If your query hava a result set the data will be displayed in the Data tab below. Here you can view data/change data, to update database right click on datagrid and select "Save".

You can also type in several queries at one time seperated by semicolons and they will be executed in one batch.

Too see posible completions use the Tab key and to select previously executed commands use the UP and DOWN arrows.

Aliases

Aliases are predefined commands that can have optional arguments that can be executed.

Example:
alias -a en "select a.flag, a.title, b.title as 'norwegian title', a.title_no from asystitlesen a, asystitlesno b where a.title LIKE '%{0}%' and a.title_no = b.title_no"
will add a alias "en" that you can use like this: "en workflow" will display all titles that contains the word workflow in both english and norwegian.
You can also manage your aliases from the Tools | Aliases menu.

Built-inn commands:

  • ls -t [%table name%], list all tables in db opptionally filtered by %table_name%
  • ls -c [%table name%] [%column name%], list all columns in db optionally filtered by %table name% and %column name%
